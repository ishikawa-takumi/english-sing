# Sing Script Toggle Reader

A Streamlit app to learn English with dialogues from the movie "Sing" (2016), with tap-to-toggle translations.

## Features

- **Tap-to-Toggle Lines**: Tap any sentence to switch that line between English and Japanese
- **Scene Filter**: Filter dialogues by scene
- **Character Filter**: Filter by character (Buster Moon, Johnny, Rosita, etc.)
- **Search**: Search for specific words or phrases
- **Reset All**: Return all lines to English with one click

## Installation

```bash
pip install -r requirements.txt
```

## Usage

Run the app:

```bash
streamlit run app.py
```

The app will open in your browser at `http://localhost:8501`.
