# Sing Movie Script Checker

A Streamlit app to learn English with dialogues from the movie "Sing" (2016).

## Features

- **List View**: Scrollable list of movie dialogues
- **Scene Filter**: Filter dialogues by scene
- **Character Filter**: Filter by character (Buster Moon, Johnny, Rosita, etc.)
- **Japanese Translation Toggle**: Show/hide Japanese translations
- **Search**: Search for specific words or phrases

## Installation

```bash
pip install -r requirements.txt
```

## Usage

Run the app:

```bash
streamlit run app.py
```

The app will open in your browser at `http://localhost:8501`.
